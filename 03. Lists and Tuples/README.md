# 03. Lists and Tuples

Practice files covering Python lists and tuples, following the Apna College curriculum.

## Files

| File | What it covers |
|------|----------------|
| `01_List.py` | List basics — creating, indexing, slicing |
| `02_List_Methods.py` | Built-in list methods (append, insert, remove, etc.) |
| `03_Tuple.py` | Tuple basics — creating, indexing, immutability |
| `04_Tuple_Methods.py` | Built-in tuple methods and operations |
| `05_Practice01.py` – `08_Practice04.py` | Practice exercises applying list/tuple concepts |
| `09_Tuples_Lists_Mini_Project.py` | Mini project — Student Grade Tracker using tuples and lists |

## Concepts covered
- Lists (mutable) vs. Tuples (immutable)
- Indexing and slicing
- List methods: `.append()`, `.insert()`, `.remove()`, `.extend()`
- Tuple unpacking
- `max()` / `min()` with `key=lambda`

## Mini Project: Student Grade Tracker
A console-based project that stores students as `(name, marks)` tuples in a list, then finds the highest/lowest scorer, adds/removes students, and counts totals.

**Learning outcomes:**
- Understood mutable vs immutable data structures
- Learned how `max()`/`min()` compare tuples and how `key=` changes that
- Practiced clean, PEP8-friendly code
