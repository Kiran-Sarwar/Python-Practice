#  07. File Input & Output in Python

This folder contains my practice programs for **File Input & Output (I/O)** in Python. These exercises helped me understand how to work with text files, including creating, reading, writing, and appending data.

##  Topics Covered

- Opening files using `open()`
- Reading file contents
  - `read()`
  - `readline()`
  - `readlines()`
- Writing to files using `write()`
- Appending data using `append` mode
- File modes
  - `r` → Read
  - `w` → Write
  - `a` → Append
  - `x` → Create
  - `r+` → Read & Write
- Closing files using `close()`
- Using the `with` statement for automatic file handling
